# Sensu::Transport

[![Code Climate](https://codeclimate.com/github/portertech/sensu-transport.png)](https://codeclimate.com/github/portertech/sensu-transport)

## Installation

Add this line to your application's Gemfile:

    gem 'sensu-transport'

And then execute:

    $ bundle
